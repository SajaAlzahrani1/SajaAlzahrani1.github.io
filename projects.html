My Projects

Here are some of the projects I've worked on:

◆ Mueen - Medication Tracker App

Course: System Analysis and Design - April 2025

This project focuses on a comprehensive system designed to track medication doses. It includes a Software Requirements Specification (SRS),
a Software Design Document (SDD), and a UI prototype. The system features a user-friendly interface that sends timely medication reminders to ensure
patient adherence. Additionally, it provides a connected interface for doctors to monitor adherence and receive real-time updates on their patients' health status.

◆ Ice Cream Ordering App

Course: Fundamentals of Flutter Mobile App Development - July 2025

This project was developed as part of a short fundamentals course in mobile application development. The app was built using Flutter,
with Dart as the programming language. It is a simple ice cream ordering application that focuses on basic UI design, navigation, and user interaction.

◆ Connect 4 – Machine Learning Move Prediction

Course: Machine learning - November 2025

In this project, my team and I developed an AI agent for the Connect 4 game. The objective was to identify the most effective algorithm for predicting
the best next move by evaluating different machine learning approaches. We implemented and compared four models: XGBoost, Ensemble Models with
Heuristic Features, Stacked Ensemble using Blending, and Convolutional Neural Network (CNN). After analyzing the accuracy of each model, we identified
the best-performing one to optimize the agent's winning strategy.
